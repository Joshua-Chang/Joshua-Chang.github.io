RecyclerView奇技淫巧：添加Item

2017-05-17 1 min read # Design # RecyclerView # Item

  1. 在adapter中新建一个方法，为数据源集合添加数据

    public void addItem(int position){
        mStrings.add(position,"add"+position);
        notifyItemInserted(position);
    }

  2. 在使用adapter时调用

    adapter.addItem(3);

  • 当然也有其他刷新adapter的方法

    notifyItemRemoved(position)
