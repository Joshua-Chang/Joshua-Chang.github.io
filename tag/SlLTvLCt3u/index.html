dagger2的Qualifier与Scope

Qualifier即Named

当module的@Provides提供相同变量的不同属性时：用于区分把哪一个初始化

Module
@Module
public class ClothModule {
    @Blue
    @Provides
    public Cloth getBluecloth(){
        Cloth mCloth=new Cloth();
        mCloth.setColor("蓝");
        return mCloth;
    }
    @Named("Red")
    @Provides
    public Cloth getRedcloth(){
        Cloth mCloth=new Cloth();
        mCloth.setColor("红");
        return mCloth;
    }
}
